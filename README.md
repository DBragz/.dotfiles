# .dotfiles

## PowerShell

Initial scripts to set up my PowerShell environment.

### Prerequisites

1. Open up PowerShell as **Administrator**.

2. Run the following command.

```shell
Set-ExecutionPolicy AllSigned
Set-ExecutionPolicy RemoteSigned -scope CurrentUser
```
